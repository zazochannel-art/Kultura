# Kultura — Web preview

This directory contains a simple static HTML/CSS preview of the Login/Dashboard UI. It's a fast preview while the full Compose Multiplatform port is prepared.

Run locally (PowerShell):

```powershell
cd web_preview
.\run-web.ps1
# then open http://localhost:8000 in your browser
```

If you want the full Compose Multiplatform implementation (real Compose UI in browser), I can scaffold a `web` Gradle module that targets Kotlin/JS + Compose Web — confirm and I'll add it next.
